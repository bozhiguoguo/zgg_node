1、商标驳回复审是商标注册被驳回后，商标申请人唯一救济途径。

2、商标局在审查商标时，仅从商标层面进行分析判断。而商标驳回复审时，不仅考虑商标本身，还会根据驳回复审理由书及相关证据材料进行评判，综合考虑商标申请人的影响力、品牌影响力，商标设计理念、商标含义等综合因素进行评审，这会大大提高商标的成功率。

3、商标局审查工作是审查员依据《商标法》、《商标审查标准》而做出主观判断为唯一的审查方式，受商标审查人员综合素质影响较大。而商标驳回复审时商标评审委员以3人以上小组平定制进行审核，因此受到个人主观判断因素影响相对较小。

1、商标法规定当商标权变化时必经法律程序，否则商标将被撤销。

2、不经商标局备案而产生的的私下商标交易，商标权利出现真空，容易导致商标权无效

3、如发生权利转移却没有进行转让程序，商标无法进行无形资产的评估及转化

4、如发生权利转移却没有进行转让程序，商标不能续展，也无法获得品牌荣誉（如著名、驰名商标）

根据《商标法》的规定，商标进行名义或地址变更时，必须一并变更。也就是说，哪怕申请人名下有一百件商标，要变更就必须一起变，如果只变九十九件商标，商标局也会要求将所有商标一并变更，否则商标局拒不接受变更申请。

商标变更中，如果申请人名义与地址都进行变更，可一并提交。但是费用需要以两件商标变更计费。

商标根据所变更的内容不同分为以下几种

1、 名义变更：商标申请人主体没变，只是申请人主体的名称变了。

2、 地址变更：申请商标时的地址与现在的营业执照（个人是身份证地址或个体工商户地址）地址不一致。

3、 删/减商品服务项目：商标中部分商品服务项目不再需要了而产生的变更。

4、 商标代理人/文件接收人变更：更换商标代理组织。

1、延续商标权，持续获得法律保护

2、延续不断的积累商标的无形资产价值，获得更高的评估价值、

3、只有不中断持续使用、知名度高的注册商标才有资格认定著名商标和驰名商标。

4、由于《商标法》的不断变化，部分商标依照新《商标法》已无法注册成功，只能通过续展来延续商标权利。例如中华、长城、北京、两面针等品牌皆属于此。

商标使用许可是指商标注册人通过商标局备案的法定程序允许他人使用其注册商标的行为。通俗解释为将商标“借”给他人使用，而进行许可备案商标许可包括普通许可、排他许可、独占许可。

个人名义申请商标，转为公司名下，并不是商标变更手续，而是商标转让手续

商标变更指的是申请人的主体本身没有改变，只是名字变了（例如：北京知果科技有限公司变为北京知果果技术有限公司）。而个人名下商标变为公司名下，哪怕原商标申请人是企业法人代表，在法律意义上也属于不同的法律主体，因此只能进行商标转让。（或者商标许可备案）

1、商标变更申请人签字或盖章的商标代理委托书

2、商标变更申请人签字或盖章的商标变更申请书

3、工商部门出具的变更证明原件或盖鲜章的复印件（变更注册人名义）、营业执照复印件（变更地址）；

1、填写版权变更申请表。

2、已登记的版权证书复印件。

3、变更登记相关的其它材料。

1、填写作品登记申请书、作品登记表、权利保证书各一份，提交作品原件及复印件、作品说明书（说明创作构思、作品主要特点及内容等）各一份；

2、申请人身份证明；

3、合作作品申请登记的，还应提交合作作者的身份证复印件（合作作者是单位的，应提交营业执照或法人代码证的复印件、法人代表的身份证复印件）、合作创作合同或协议的原件及复印件各一份；

4、委托创作作品申请登记的，著作权人还应提交著作权人及创作者的身份证复印件（著作权人或创作者是单位的，应提交营业执照或法人代码证的复印件、法人代表的身份证复印件）、委托创作合同或协议的原件及复印件各一份；

5、职务作品申请登记的，应提交作者的身份证复印件、著作权人或专有使用权人的营业执照或法人代码证的复印件、法人代表身份证复印件、聘用合同及著作权归属证明原件及复印件；

6、美术作品或摄影作品申请登记的，申请人除按上述情况提交有关材料外，还需提交作品照片两张（不大于3r，可电脑打印），一张贴在“作品登记申请书”左下方空白处，并在上面盖骑缝章，另一张随材料提交。

专利保护的范围是以专利文件中权利要求书划定的范围为准的，保护的内容也就是权利要求书公开的内容。

1、保护自己的技术成果不受他人侵犯

2、防止他人滥用专利权恶意、主动造成侵权行为

3、办理高新技术企业、申请政府补贴、资助等必备资料

4、提高企业核心竞争力、保护企业产品市场占有份额

5、形成企业知识产权资产，提升企业注册资本、企业价值 。

可以在国家知识产权局官网专利检索网页输入关键词进行检索，实用新型和外观专利只有授权才能公开，发明专利在提交之后6-18个月才会公开，因此没有公开的专利文件是检索不到的。

申报专利的技术方案和已有的技术方案存在区别点，并且该区别点会带来有益效果才会有可能被授权，已有的技术方案包括已申报的专利、已经公开的文献、视听材料等。发明专利是将已有技术结合起来与申请的技术方案作对比，仍有区别点，且区别点能够带来有益效果，才有可能会被授权，实用新型和外观是一对一的与现有技术做对比，只要有区别点，且区别点会带来有益效果就会被授权，例如现有技术中风扇需要手动才能够实现转动，现在将手动的结构改进成电机带动风扇转动的结构，这样电动的结构与手动的结构存在区别点，并且带来了能够省力的有益效果。

产品外观的新设计：例如造型独特、图案原创设计的灯具、餐桌、办公椅、玩具等各类生活用品以及工厂生产机器等等。

1、产品结构：例如电脑的硬件结构都是由哪些器件组成的、生产设备结构都是由哪些部件组成的、生物检测装置结构都是由哪些构件组成的等等；

2、方法类：例如网站、APP、窗口等制作方法；化合物合成方法、检测方法等；细胞培养、扩增等方法；污泥处理、垃圾处理等方法；产品制造、加工方法等等；

3、用途类：例如发现了杀虫剂还具有除草的用途；感冒药还能用于缓解心脏疼痛的用途等。

国家知识产权局在审查时，会将为什么不通过的理由以书面的形式下发至申请人，申请人如果认为审查员理解的有错误、理由不准确等，可以通过驳回复审的途径使专利再次进行审查。

首先，需要申请人提供自己的技术方案，专利工程师做出检索，判断新创性，申请人根据新创性确定是否要申报专利，如果申报的话，提供申请人等信息，服务款项到后，专利工程师开始撰写专利，7-15天会给申请人发专利初稿，申请人与专利工程师对专利稿件达成一致，定稿确认后提交到国家知识产权局，国家知识产权局开始审查工作，审查过程中审查员如认为专利稿件有不能授权的地方，专利工程师会帮助申请人进行答辩，直至审批结果下来

优先权是指，当技术材料不完整时，可以先将这份不完整的技术资料提交的国家知识产权局获得申请日，在12个月内将技术资料补充完整后再提交到国家知识产权局，以此避免申请人将设计思路转化成具体技术方案这段时间内，其它人申请同样或类似的技术方案对申请人会产生不予授权的情况。

专利费用减缓是指交给官方的官费是可以办理费用减缓的，可以理解为官方给予亏损的公司、低收入的个人费用的折扣优惠，公司申请费用减缓的话，需要当地科委或国家知识产权局开具同意费用减缓的证明。

只有申报发明专利中产品的结构可以申报实用新型专利：例如，电脑的硬件结构都是由哪些器件组成的、生产设备结构都是由哪些部件组成的、生物检测装置结构都是由哪些构件组成的等等。
